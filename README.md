# cruise-adventure-cruise · Claude Code 插件

为喜欢冒险的旅行者推荐刺激的邮轮产品，体验不同寻常的旅程。

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/cruise-adventure-cruise-claude-plugin
/plugin install cruise-adventure-cruise@cruise-adventure-cruise-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **cruise-adventure-cruise**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
